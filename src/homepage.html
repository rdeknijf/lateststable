Fetch the latest stable version of anything

This is a free JSON REST API service to figure out what the latest stable version of something is.

Try it!

  • https://lateststable.org/v1/pypi/requests
  • https://lateststable.org/v1/jetbrains/pycharm
  • https://lateststable.org/v1/docker/bitnami/redis
  • https://lateststable.org/v1/github/kubernetes/ingress-nginx
  • https://lateststable.org/v1/wikipedia/Linux
  • https://lateststable.org/v1/npm/axios

There's currently six endpoints:

  • Pypi
  • Github
  • Docker
  • Wikipedia
  • Jetbrains
  • NPM

The result will always be in this format:


{
  "name": "somepackage",
  "version": "3.2.1",
  "platform_version_string": "3.2.1",
  "major": "32",
  "minor": "2",
  "patch": "1"
}
